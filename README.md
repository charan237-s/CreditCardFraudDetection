# 💳 Credit Card Fraud Detection System

## 📌 Project Overview

This project detects fraudulent credit card transactions using Machine Learning.

## 📂 Dataset

- Total Transactions: 284,807
- Features: 30
- Target: Class
  - 0 → Genuine
  - 1 → Fraud

## 🚀 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- XGBoost
- Streamlit
- Joblib

## 🤖 Machine Learning Models

- Logistic Regression
- Random Forest
- XGBoost

## 📊 Project Features

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Scaling
- SMOTE
- Model Comparison
- Fraud Prediction
- Streamlit Web Application

## 📈 Results

- Logistic Regression Accuracy: 97.36%
- Random Forest Accuracy: 99.95%
- XGBoost Accuracy: 99.92%
